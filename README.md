# Fishing-with-DeeR

라즈베리파이를 이용해서 만든 낚시 게임입니다.

낚싯대 그립을 직접 제작해 MPU 9250 센서와 Grove - Haptic Motor 를 부착하여 낚시 모션을 감지하고, 낚시 게임이 시작되어 물고기가 미끼를 물었을 때 진동을 주어 현실감 있는 게임을 만들고자 했습니다.

MPU 9250 모듈을 사용해 가속도와 기울기를 측정하여 낚시 모션을 취하면 이를 감지하여 클라이언트에 신호를 전송하는 원리로 낚시 게임을 시작합니다.
또한 데이터베이스를 생성하여 도감 관리, 어항 관리, 낚싯대 강화, 랭킹 관리 등의 작업을 수행했습니다.

웹 화면에서는 낚시터 화면과 도감 화면, 어항 화면, 상점 화면, 랭킹 화면 등을 구성하여 게임의 재미를 높이기 위해 노력했습니다.
